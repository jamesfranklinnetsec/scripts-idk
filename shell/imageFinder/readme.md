# imageFinder

Finds all files of type image as per the file command.

## Installation

copy shell scripts into directory you would like to find all images in and 
```bash
cp *.sh ~/path/to/destination
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'

foobar.singularize('phenomena') # returns 'phenomenon'
```
